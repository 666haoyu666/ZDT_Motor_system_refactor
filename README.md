# ZDT Motor System Refactor

STM32 firmware project for the ZDT motor system.

## Project Layout

- `00_Doc/` - project documents
- `01_App/` - application layer
- `02_Service/` - service layer
- `03_Middleware/` - middleware adapters
- `04_Bsp/` - board support package
- `Core/` - STM32CubeMX generated core files
- `Drivers/` - STM32 HAL and CMSIS drivers
- `MDK-ARM/` - Keil MDK project files
