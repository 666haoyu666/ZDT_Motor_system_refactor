欧氏距离场安全检测

把地图变成距离查表图，再用平滑路径采样点逐个判断是否安全。
障碍物 小于安全距离 临界区域 安全余量较大 平滑路径

第五步到底在做什么

距离场不是再找一条路，而是给每个位置算一个值：当前位置离最近障碍物还有多远。平滑路径出来以后，把曲线按固定间距采样，每个采样点去距离场里查表。

dist_map[x][y] = min(distance((x, y), obstacle))
is_safe = dist_map[path_point] >= robot_radius + safety_margin

如果查出来的距离小于安全阈值，这个采样点就标红，说明样条曲线可能“抹进”障碍物附近。

工程上怎么用

  1. 先在障碍地图上计算距离场，单位最好换成毫米。
  2. 对 B 样条或三次样条曲线做等距采样。
  3. 每个采样点用双线性插值查 dist_map。
  4. 只要有采样点小于 robot_radius + safety_margin，这段平滑路径就不合格。
  5. 不合格时，回退局部平滑、增加控制点，或降低平滑强度。
float safe_dist = robot_radius + safety_margin;
bool path_safe = true;

for (int i = 0; i < path_count; i++)
{
    float d = DistanceField_Bilinear(dist_map, path[i].x, path[i].y);

    if (d < safe_dist)
    {
        path_safe = false;
        MarkUnsafePoint(i);
    }
}

读图要点

  • 黑色格子是障碍物。
  • 红色区域表示离障碍小于安全阈值。
  • 蓝色曲线是平滑后的路径，圆点是等距采样输出点。
  • 圆点变红时，说明平滑路径需要回退或加控制点。
