方法取舍

方法 是否经过关键点 优点 主要风险
A* 原始折线 经过全部格点 可行性强，便于调试 折角硬，点数多
Shortcut 折线 经过保留关键点 简单有效，能直接减少冗余点 仍然是折线，拐角不连续
三次样条 经过关键点 路径连续，控制目标明确 局部过冲后可能靠近障碍
三次 B 样条 通常不经过中间点 平滑度高，曲率变化更温和 曲线可能从障碍物边界内侧抹过去
Bezier 分段时可经过端点 局部形状容易控制 长路径整体处理不如样条自然
五次多项式 通常只约束起终点 适合速度、加速度连续的轨迹段 不是纯几何路径平滑的首选

落地建议

  • A* 搜索前先膨胀障碍物，给后续平滑留下安全余量。
  • 安全距离现在同时参与 A* 规划和后续检测；小幅调节可能仍保持同一条离散最优路径。
  • Shortcut 后保留的点是曲线控制点，不是最终输出点。
  • 任何样条平滑之后都要重新做碰撞检测，不能只看曲线是否好看。
  • 最终给控制层的路径建议使用等距采样点，并补充 yaw 和曲率。
